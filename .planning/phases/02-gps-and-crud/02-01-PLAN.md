**gas/Code.gs** — Add three functions after the existing `jsonResponse` function:

1. `doPost(e)` function:
   - Parse body: `var body = JSON.parse(e.postData.contents);`
   - Route by `body.action`: `'add'` calls `addToilet(body.data)`, `'update'` calls `updateToilet(body.data)`
   - Unknown action returns `jsonResponse({ status: 'error', error: 'Unknown action' })`

2. `addToilet(data)` function:
   - Get sheet: `SpreadsheetApp.getActiveSpreadsheet().getSheetByName('toilets')`
   - Generate ID: `var id = sheet.getLastRow();` (simple auto-increment)
   - Generate timestamp: `var now = new Date().toISOString();`
   - Append row with column order: `[id, data.buildingName, data.floor || '', data.locationMemo || '', data.hasLock || false, data.password || '', data.latitude, data.longitude, data.region || 'misa new town', data.category || '', now, now]`
   - Return `jsonResponse({ status: 'ok', data: { id: String(id), createdAt: now } })`

3. `updateToilet(data)` function:
   - Get sheet and all rows via `sheet.getDataRange().getValues()`
   - Loop from row index 1, find row where `String(rows[i][0]) === String(data.id)`
   - Update only provided fields using `sheet.getRange(row, colIndex).setValue(value)` — column mapping: buildingName=2, floor=3, locationMemo=4, hasLock=5, password=6, category=10
   - Always update updatedAt (column 12) to `new Date().toISOString()`
   - Return `jsonResponse({ status: 'ok', data: { id: String(data.id), updatedAt: now } })`
   - If ID not found: `jsonResponse({ status: 'error', error: 'Toilet not found: ' + data.id })`

Use `var` for all declarations (GAS Apps Script compatibility — per Phase 1 convention).

Create new file `src/js/geo.js` with four exported functions:

1. `export function getCurrentPosition()`:
   - Returns a Promise
   - Check `navigator.geolocation` exists, reject with `new Error('Geolocation not supported')` if not
   - Call `navigator.geolocation.getCurrentPosition(success, error, options)`
   - Options: `{ enableHighAccuracy: true, timeout: 10000, maximumAge: 30000 }`
   - Resolve with `{ lat: position.coords.latitude, lng: position.coords.longitude, accuracy: position.coords.accuracy }`
   - Reject with the geolocation error object on failure

2. `export function haversineDistance(lat1, lng1, lat2, lng2)`:
   - Earth radius: `var R = 6371000;` (meters)
   - Convert degree differences to radians: `(lat2 - lat1) * Math.PI / 180`
   - Standard Haversine formula: `a = sin(dLat/2)^2 + cos(lat1)*cos(lat2)*sin(dLng/2)^2`
   - `c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a))`
   - Return `Math.round(R * c)` (distance in meters, rounded to integer)

3. `export function formatDistance(meters)`:
   - If `meters < 1000` return `meters + 'm'`
   - Else return `(meters / 1000).toFixed(1) + 'km'`

4. `export function findNearest(userLat, userLng, toilets)`:
   - Iterate over toilets array
   - Calculate distance for each using `haversineDistance(userLat, userLng, toilet.latitude, toilet.longitude)`
   - Attach `toilet._distance = dist` to each toilet object for display purposes
   - Track minimum distance and corresponding toilet
   - Return the nearest toilet object (or null if toilets array is empty)
  </action>
